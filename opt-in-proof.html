Luna Messaging Opt-In Process Luna Messaging is not open to the public and does not use purchased lists, lead lists, or mass signup forms. How users opt in: 1. The user directly asks Brett Hoffman for access to Luna messaging. 2. The user provides their mobile number and explicitly requests messaging access. 3. The number is manually reviewed and added to an approved allowlist. 4. Luna only sends messages after approval, either in response to the user’s inbound message or as a reminder/utility message specifically requested by the user. 5. Users can reply HELP for help or STOP to opt out.
